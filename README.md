# Course-Registration-System
Java-based Course Registration System using OOP concepts, Java Swing GUI, exception handling, and backend service management developed for the CSE282 Java Lab project.

# Course Registration System

The Course Registration System is a Java-based desktop application developed using Object-Oriented Programming (OOP) concepts and Java Swing GUI. The system is designed to manage students, courses, and course registrations efficiently through a simple and user-friendly interface.

This project demonstrates all four OOP principles:

* Encapsulation
* Inheritance
* Abstraction
* Polymorphism

The application also includes:

* Exception handling using try-catch blocks
* Custom exception classes
* Backend service layer for data management
* Java Swing graphical user interface
* Modular package-based project structure

Features of the system include:

* Add and manage student information
* Add and manage course information
* Register students into courses
* Validate user inputs
* Display success and error messages
* GUI panels for different operations

Technologies Used:

* Java
* Java Swing
* IntelliJ IDEA
* GitHub

Project Structure:

* model package for entity classes
* service package for backend logic
* exception package for custom exceptions
* gui package for frontend panels

This project was developed as part of the CSE282 Introduction to Programming Language II (Java) Lab course project.

# How to Compile and Run

## Using IntelliJ IDEA

1. Open the project in IntelliJ IDEA
2. Wait for IntelliJ to load the project files
3. Open `Main.java`
4. Click the green ▶ Run button
5. The Course Registration System GUI will open

---

## Using Terminal / Command Prompt

### Step 1: Open project folder

Navigate to the `src` folder.

### Step 2: Compile the Java files
